## SenatsverwaltungFinanzen_Challengeaufgaben

**Executive Summary:**
Diese Analyse untersucht die langfristige Entwicklung der Zahlfälle der Eingliederungshilfe (EGH, Produktnummer 80992 – EGH SGB IX Leistungsgewährung) in den Berliner Bezirken. Ziel ist es, strukturelle Trends zu identifizieren und die Entwicklung einzelner Bezirke systematisch mit dem gesamtstädtischen Referenzwert zu vergleichen.

Zentrale Ergebnisse:
- Die Zahlfälle der EGH zeigen auf Gesamtberliner Ebene ein moderates, aber stabiles Wachstum.
- Im festen Betrachtungsfenster 2022–2024 weist Neukölln mit rund +6 % die höchste Wachstumsrate auf und liegt damit signifikant über dem Berliner Durchschnitt.
- Die Rolling-Analyse (3 Jahre) bestätigt, dass mehrere Bezirke (u. a. Neukölln und Mitte) dauerhaft überdurchschnittlich wachsen, während andere Bezirke konsistent unter dem Referenzwert liegen.
- Die beobachteten Unterschiede sind strukturell stabil und nicht durch kurzfristige Jahresschwankungen erklärbar.

**Entwicklungsumgebung:**
- IDE: Jupyter Notebook in Visual Studio Code (VSC)
- Programmierungsprache: Python 3.12.4, inkl. gängigen Bibliotheken der Datenanalyse in Virtual Enviroment (.venv)
